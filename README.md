# vim-glue

This repository contains the syntax and indentation of the configuration file of [GlueLang](https://github.com/ryuichiueda/GlueLang/).

## Installtion

### dein

```vim
dein#add('CORDEA/vim-glue')
```

or

```toml
[[plugins]]
repo = 'CORDEA/vim-glue'
```

### NeoBundle

```vim
NeoBundle 'CORDEA/vim-glue'
```
